# UDA-Bench
Unstructured Data Analysis using LLMs: A Comprehensive Benchmark [Experiments &amp; Analysis]

UDA-Bench is the first benchmark for unstructured data analysis that offers high-quality, large-volume, and diverse datasets as well as rich query workload to thoroughly evaluate unstructured data analysis systems.
